# Hotel Reservation System

A Hotel Reservation System developed in Python as part of the Programming for Data Engineering course.

## Features

- User Authentication
- Room Management
- Guest Management
- Reservation Management
- Check-in / Check-out
- Billing System
- SQLite Database
- Dashboard Interface

## Technologies

- Python
- SQLite
- CustomTkinter / Tkinter
- Object-Oriented Programming (OOP)

## Project Structure

```
hotel_reservation_system/
│
├── assets/
├── data/
├── hotel_reservation/
├── main.py
└── requirements.txt
```

## Installation

```bash
git clone https://github.com/USERNAME/hotel-reservation-system.git

cd hotel-reservation-system

pip install -r requirements.txt

python main.py
```
